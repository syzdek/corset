Design Goals
============

Kernel Sub-system Integration:
   [ ] iptables/ip6tables (firewall)
   [ ] ipset (firewall)
   [ ] nfacct (flow accounting)
   [ ] tc (traffic policer/controller)
   [ ] iproute2 (routing)

User Space Integration:
   [ ] Syslog
   [ ] Files (local provisioning templates)
   [ ] RADIUS
   [ ] DHCPD
       [ ] ISC DHCP
       [ ] Kea DHCP
   [ ] LDAP
   [ ] SQL
       [ ] iODBC
       [ ] MariaDB
       [ ] Postgres
       [ ] SQLite
